# Echonera Resale Marketplace

This repository hosts the GitHub Pages site for Echonera, a premium resale marketplace.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (GitHub Pages)
- **E-commerce**: Shopify Starter
- **Domain**: GoDaddy

## Setup Instructions

### GitHub Pages
1. Push code to GitHub
2. Enable GitHub Pages in repository settings
3. Site will be available at: https://ari-xanna.github.io/Echonera.github.io/

### Shopify Integration
- Embed Shopify Buy Buttons or link to your Shopify store
- Configure domain in Shopify settings

## Development
Edit files in VS Code and push changes to GitHub to update the live site.
